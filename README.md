# SweetIDE - CodeEditing

- 설치 매뉴얼
  - yarn을 설치해주세요.
  - `yarn` 을 입력해서 의존성 모듈 설치

- 명령어
  * `yarn start` - Dev용 서버 실행
  * `yarn run build` - 빌드
